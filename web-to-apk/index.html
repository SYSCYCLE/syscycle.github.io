Nasıl Çalışır?

    2. 
      
  2. 2 — Ayarla: uygulama adı, paket adı, ikon, renkler, yönelim, izinler ve WebView davranışları.
    3. 
      
  3. 3 — Önizle: telefon çerçevesinde uygulamayı anında gör.
    4. 
      
  4. 4 — Üret: düğme; MainActivity.java, AndroidManifest.xml, Gradle dosyaları, tüm yoğunluklarda launcher ikonları, splash ekranı, PWA (manifest + service worker) ve GitHub Actions iş akışını içeren tam bir Android Studio projesi oluşturup ZIP olarak indirir.
    5. 
      
  5. 5 — Derle:
        
      
          
    • Android Studio → Build → Build APK(s)
      • 
          
    • Terminal → gradle wrapper + ./gradlew assembleDebug
      • 
          
    • GitHub'a push et → Actions otomatik APK üretir (bilgisayara gerek yok!)
